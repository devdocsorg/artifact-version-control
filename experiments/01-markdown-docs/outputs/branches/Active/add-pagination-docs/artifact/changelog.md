# Changelog

## v2.1.0 — 2025-01-15

### Added
- Project endpoints (`/projects`)
- OAuth 2.0 support

### Changed
- Rate limit increased from 500 to 1000 requests/minute

## v2.0.0 — 2024-11-01

### Breaking Changes
- Base URL changed from `/v1` to `/v2`
- User response format changed: `username` field renamed to `name`

### Added
- User creation endpoint
- API key rotation endpoint

## v1.0.0 — 2024-06-01

### Initial Release
- User listing and retrieval
- API key authentication
- Basic error handling
